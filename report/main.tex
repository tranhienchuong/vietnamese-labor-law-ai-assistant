\frontmattersection{ABSTRACT}
This work presents a Vietnamese labor-law question-answering assistant built around citation-grounded retrieval. The system preserves statutory hierarchy, combines Qdrant hybrid retrieval with a Neo4j legal graph, validates answer citations deterministically, and exposes the assistant through a FastAPI backend and a React/Vite frontend with Supabase authentication and data storage.

In the deployed runtime, the system uses a configured LLM provider, including GPT-OSS 120B, to generate grounded answers from selected legal contexts. For reproducible benchmarking, the final evaluation uses deterministic extractive generation to avoid provider variability. On a 100-query benchmark, graph-augmented retrieval improves Recall@10 from 0.735 to 0.835 and the final end-to-end run obtains an adjusted pass rate of 0.780, citation validation pass rate of 1.000, and out-of-corpus QA pass rate of 1.000. These results show strong citation containment and refusal behavior, while retrieval robustness remains the main limitation. The system is a scoped legal information assistant, not a legal advice tool.
